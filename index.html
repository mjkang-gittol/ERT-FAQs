문장을 AI로 더 정교하게 수정할 수 있나요?AI
콘텐츠 생성 후 우측 하단 [AI로 재작성] 버튼을 통해 가능합니다.
과거 버전과 현재 내용을 비교할 수 있나요?
[버전 관리] > [비교 및 교체] 메뉴에서 수정된 부분을 하이라이트로 확인 가능합니다.
도식, 표, 그래프
보고서에 차트를 어떻게 추가하나요?
본문 작성란의 [+] 버튼 > [차트]를 클릭하여 삽입합니다.
차트 데이터 수치를 수정하려면 어떻게 하나요?
[차트 데이터 편집] 탭에서 테이블 형식으로 수치를 직접 수정할 수 있습니다.
이미지 차트와 HTML 기반 차트의 차이점은 무엇인가요?AI
HTML 기반 차트는 텍스트 검색과 실시간 AI 번역이 가능하여 유지보수가 매우 용이합니다.
엑셀(Excel) 데이터를 일괄 가져올 수 있나요?
네, 엑셀 수치를 복사하여 데이터 테이블에 직접 붙여넣기(Copy & Paste) 할 수 있습니다.
